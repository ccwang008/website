知道了

随着移动互联网的快速发展，LTE 用户呈直线上升趋势，用户的多种高速数据业务需求给LTE 网络的建设维护优化提出了更高的要求，无线环境的多变性和复杂性带来的业务感知通过网络评估分析经验显示，用户观看1080p 视频和初始缓冲时延小于3s 对于小区下行单用户平均感知速率至少要达到10Mbps 水平。


本模型以下行单用户平均感知速率为依据，结合提供的基站工程参数，天线指标和各类无线网络统计指标采用相关的人工智能方法探知影响用户感知速率的因素，分析存在下行单用户平均感知速率小于10Mbps 的低速小区具体情况，形成网络质量感知分析预测体系。


使用方法该模型实现了1 个API 接口，其调用形式和返回值格式如下：1.predict ：根据LTE 无线网络的分类统计指标和参数，预测用户感知速度。



 HTTP 方法：POST


 模型方法：预测


 HTTP 请求体格式：{“ lte_specs” ：长度为426 的特征列表}


 HTTP 响应体格式：{“ value”：是否达到10Mbps 预测结果}
